# KakaoStory-Based-RestAPI
- Kakao WEB API 

## Description
- 카카오스토리 웹 공식 버젼이 나오기 전 Kakaostory Rest API 기반 API 를 공개했을 당시 개발 되었던
KakaoStory on Web 프로젝트 입니다. 

## Feature 
- 글 쓰기 (사진 첨부 포함)
- 프로필 사진 변경

## Purpose
- 모바일에서만 되었던 불편함을 해소하기 위해 개발 된 웹 기반 프로젝트.
- PC 에서도 카카오스토리에 글을 쓸 수 있게 하기 위해서
